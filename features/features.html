ASA Stats Features

Tools for Tracking, Evaluating, and Acting on Your Algorand Portfolio

ASA Stats provides a full-stack solution for ASA and NFT management — from wallet overviews to advanced DeFi analytics and trading tools. Whether you’re a casual user or a professional builder, these features give you real-time visibility and control.

Live Features

These features are fully developed and available to all users (some with tiered access):

Portfolio Tracker

ASASTATS is used for:

• View full ASA & NFT holdings for any Algorand address
• Track values in ALGO, USD
• Analyze wallets, liquidity pools, staked funds, and more — all in one place

User Bundles

• Combine multiple addresses (or .algo names) into one portfolio view
• Create up to 38 bundles (Cluster tier); 3 bundle always available for free users

Real-Time ASA & NFT Valuation

• Fetch live prices from DEXs like Tinyman, Pact, and Humble
• Consolidate data from swaps, vaults, and liquidity pools
• NFT collections auto-aggregated, with floor and estimated value

API v2 Access

• Get programmable access to the same valuation engine used on-site
• Query bundles, addresses, and NFTs via REST API (JSON/XML/YAML)
• Available to Asastatser+ tiers and all DAO governors

CSV Export

• Export transactions by address, wallet group, or bundle
• Filter by asset type, protocol, or value threshold
• Planned tax-compliant CSV formatting under development

In Beta / Preview Mode

Historical Portfolio Charts

• Visualize portfolio value over time
• Currently rolling out as “Historic Data” widget

NFT Pricing Model (AI-assisted)

• Beta model trained on real market data + metadata rarity
• Continually retrained based on new listing and sale patterns

Bundle Sharing (Public URLs)

• Reserve public URLs for bundles
• Public views display real-time value, allocation, and asset labels
• Part of a future release phase

Feature Requests & Community Input

ASA Stats development is driven by user demand. Most new features originate from:

• GitHub issues
• Community proposals
• Rewarded contributions (via ASASTATS tokens)

Got a feature idea or improvement request? Submit a GitHub Issue.
